# Eithar
Eithar is a graduation project implementing a time-banking system where users exchange services using time credits instead of money. The system was developed using HTML, CSS, JavaScript, Python, and MySQL, following the Waterfall methodology, with a focus on usability, system modeling, and data-driven design.

## Database Setup
This project uses MySQL as the database system.

To run the project locally:
1. Create a MySQL database.
2. Import the provided SQL schema .
3. Update the database connection settings in the configuration file.
